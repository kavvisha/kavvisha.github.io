What is DocSpace?

With the expansion of modern technology, the lifestyle of people is mostly managed and relied on their computers and mobile phones to an increasing extent. On that premise, File classification is a technique in modern storage systems and can be introduced as a hierarchical arrangement of classification levels and it allows files to be stored in the most efficient approach. DocSpace is a product to classify and secure data with the aid of security levels. It analyzes data before classifying and are categorized as Text files and Image Files. In addition, appropriate measures are taken to prohibit accessing files in lower security level to another higher security level without proper authorization. The major goal is to update from the current security level of the file after each modification to improve accuracy. DocSpace is a single integrated based hybrid data classification system which is developed using High level Microservice architecture to prevent intentional or unintentional misbehaviors.
